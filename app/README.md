# CalorieTracker — Дневник учета калорий и БЖУ

Мобильное приложение для платформы Android, разработанное в рамках производственной практики. Позволяет вести подневный учет съеденных продуктов, контролировать калорийность рациона и баланс макронутриентов.

## Функционал
* Автономный справочник продуктов (БЖУК на 100г).
* Добавление приемов пищи с автоматическим пересчетом КБЖУ под вес порции.
* Просмотр истории за любой день с помощью встроенного календаря.
* Полный цикл операций CRUD (создание, чтение, изменение, удаление).
* Полная автономность работы (Offline-режим) без интернета.

## Стек технологий
* **Язык:** Java
* **База данных:** Локальная СУБД SQLite
* **Интерфейс:** XML, CardView, Material Design
* **Минимальная версия Android:** API 21 (Android 5.0 Lollipop)

## Инструкция по запуску
1. Склонируйте репозиторий или скачайте архив с проектом.
2. Откройте папку проекта в Android Studio.
3. Дождитесь синхронизации Gradle.
4. Подключите реальное устройство через USB (с включенной отладкой по USB) или запустите эмулятор.
5. Нажмите кнопку **Run** (Shift + F10) для сборки и установки APK.
